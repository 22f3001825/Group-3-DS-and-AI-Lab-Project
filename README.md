# CS2007 MLT Learning Companion - Preprocessing Pipeline

This repository contains the dataset preprocessing, cleaning, chunking, and splitting pipeline for the Course-Aware Personalized Learning Companion (CS2007 - Machine Learning Techniques).

---

## 📁 Repository Structure

```text
.
├── data/
│   ├── raw/                         # Raw unstructured files (PDFs, FAQs, Notes)
│   ├── processed/                   # OCR-extracted and Markdown-standardized documents
│   ├── cleaned/                     # Scrubbed documents with restored timestamp headers
│   └── splits/                      # Final LangChain JSON-L chunks for RAG ingestion
│       ├── train_chunks.jsonl       # Weeks 1-8 chunks (4,304 chunks)
│       ├── val_chunks.jsonl         # Weeks 9-10 chunks (216 chunks)
│       └── test_chunks.jsonl        # Weeks 11-12 chunks (192 chunks)
│
├── src/
│   ├── process_dataset.py           # Uses PyMuPDF4LLM & EasyOCR to extract PDFs to Markdown
│   ├── clean_dataset.py             # Normalizes whitespace, fixes OCR hyphens, converts timestamps to headers
│   └── prepare_rag_splits.py        # LangChain MarkdownHeaderTextSplitter & RecursiveCharacterTextSplitter
│
├── reports/
│   ├── document_cleaning_report.md  # Metrics generated by clean_dataset.py
│   └── chunking_report.md           # Metrics generated by prepare_rag_splits.py
│
├── requirements.txt                 # Pinned dependencies for reproducibility
├── Milestone report/                # Documentation and grading reports
└── README.md                        
```

## 🚀 Setup & Execution Instructions

### 1. Prerequisite: Setup Virtual Environment
Initialize a clean Python environment and install the pinned dependencies:

Using `uv` (Recommended):
```bash
uv venv
uv pip install -r requirements.txt
```

### 2. Run the Data Pipeline
To execute the dataset preparation from scratch, run the scripts in this exact order:

**Step 1: Extract & Standardize**
Converts all PDFs into Markdown, applying Tesseract/EasyOCR to recover embedded mathematical equations and question blanks.
```bash
uv run python src/process_dataset.py
```

**Step 2: Clean & Normalize**
Scrubs boilerplate, collapses massive blank spaces, and securely converts video timestamps into `### Timestamp: XX:XX` Markdown headers.
```bash
uv run python src/clean_dataset.py
```

**Step 3: Chunk & Split (Leakage-Free)**
Uses LangChain to chunk the dataset into exactly **384 tokens** (with a 15% overlap) and automatically extracts the timestamp headers into structured JSON `metadata`. Partitions chunks into Train, Validation, and Test sets based strictly on chronological weeks.
```bash
uv run python src/prepare_rag_splits.py
```
